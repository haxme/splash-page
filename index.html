What is HaxMe? In short, HaxMe is an online community that is focused on privacy and security. We are currently in the process of building a system that will allow us to administer security challenges in a "game-like" style. We refer to these challenges as "missions."

In the past, HaxMe offered web missions that allowed users to learn about common security vulnerabilities in web applications like SQLi, RFI, and LFI. They also enabled users to use the browser as a gateway for attacking web apps via CSRF and XSS.

So, where does HaxMe go from here? We would like to build a new generation of missions that allows users to learn about other attack vectors, in addition to some of the old challenges we offered years ago. We are currently experimenting with deploying vulnerable images to the cloud on the fly and gamifying the missions system to offer a more challenging, thrilling, and rewarding user experience.

We are currently in the design and conception phases and will present you with an official roadmap in 2019. In the meantime, please connect with us on the forums and on Discord!
